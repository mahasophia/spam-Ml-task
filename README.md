# SMS Spam Classification using Lasso Regression

## 📌 Objective
Classify SMS messages as spam or ham using TF-IDF and Lasso Regression.

## 📊 Dataset
SMS Spam Collection Dataset

## ⚙️ Steps
- Data preprocessing
- TF-IDF feature extraction
- Lasso regression for feature selection

## 📈 Results
- Total features: ~8000
- Features reduced significantly using Lasso
- Higher alpha → fewer features selected

## 🛠 Tools Used
- Python
- Scikit-learn
- Pandas
